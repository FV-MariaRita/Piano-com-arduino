# Piano-com-arduino
Projeto construído como o Trabalho 1 da disciplina de Eletrônica para Computação do curso de Ciência da Computação do 1° semestre no ICMC-USP.

## Descrição do projeto

Esse projeto consiste na construção de um piano utilizando um Arduino UNO como base para tal. O piano permite que os usuários toquem músicas utilizando as setes notas musicais (Dó, Ré, Mi, Fá, Sol, Lá, Si) organizadas de acordo com a escala maior (diatônica) das notas musicais. Ou seja, é possível tocar músicas utilizando a escala Dó - Ré - Mi - Fá - Sol - Lá - Si - Dó. 

As notas musicais são reproduzidas por meio de um buzzer e há um botão que permite que os usuários toquem cada uma das notas. Além disso, cada nota possui um LED, que funciona como indicador visual, associada a ela, de modo que, quando o botão correspondente a nota musical é pressionado, o LED acende. 

Por fim, há um nono botão na placa que ao ser pressionado faz com que uma música programada previamente seja reproduzida, bloqueando os botões que permitem que o usuário toque a música desejada.
